## ✨ Features

### 🔐 Secure Login System
- Professional authentication interface
- Demo credentials for quick access
- Session persistence

### 📊 Interactive Dashboard
- **Real-time KPI Cards**: Active members, book counts, overdue statistics
- **Dynamic Filters**: Filter by Member ID, Book ID, and Overdue Status
- **Live Data Tables**: Sortable and filterable overdue register
- **Responsive Design**: Works on desktop, tablet, and mobile

### 📈 Data Visualizations
- **Book Format Distribution Chart**: Line chart showing Science, Self-Help, and Technology collections
- **Overdue Bar Chart**: Dynamic bar chart updating with filters
- **Color-coded Status Badges**: Visual indicators for overdue status

### 📖 Built-in Documentation
- Collapsible README section inside the dashboard
- Complete user guide and feature explanation
